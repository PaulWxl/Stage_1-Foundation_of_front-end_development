易航工作室事记 2000年12月，筹划建设易航网站，几位创始人与网络中心老师取得联系，达成一致意见。 2001年3月，招聘工作人员，成立易航工作室。 2001年5月，经过一个月的紧张开发，推出易航网（第一版），并在校园引起强烈反响，受到师生普遍欢迎。 2001年5月，抓住当年全国计算机等级考试成绩公布的时机，推出民航学院计算机等级考试查询系统；许多学生第一次进入易航网，易航网的访问量于短短数日内首次突破2000大关，易航的名字开始被更多人知晓。 2001年7月，成功对易航网进行改版，受到1万多名用户的普遍欢迎。 2001年8月，开发融学生管理、信息发布、规章制度、学生就业为一体的学工部网站。 2001年9月，与党宣传部合作，成功开发民航学院（现中国民航大学）50周年校庆网站，并于校庆期间全方位、第一时间报道民航学院50周年庆典的盛大宏伟场面，受到校领导的大力支持。 2001年9月，与计算机系基础教研室合作，开发计算机系基础教研网站。 2001年10月，与设备处合作，开发学校设备调剂系统，使民航学校闲置设备得到合理利用。 2001年11月，与党委宣传部合作，开发融校园新闻、精神文明、思想教育为一体的宣传部网站。 2001年11月，参加校第三届“创新杯”创业大赛，并获得第二名的好成绩。 2001年12月，发起并举办天津高校学生网站交流会，邀请天津大学天外天、南开大学亦多网站、天津师范大学大天使之城网站负责人前来交流讨论。 2001年12月，与机电学院（相当于今航空工程学院和电子信息与自动化学院）科协合作，举办民航学院第一届网页设计大赛，营造我校科技氛围。 2001年12月，开发民航学院学生系统，为全校每位学生建立电子档案，以方便管理、查询、统计等，推动学校信息化建设。 2002年1月，与海航信息系统公司交流，易航在民航系统开始产生深远影响；举办第一届网络文明大赛，得到广大民航学子的关注，进一步推动文明上网之风。 2002年，帮助《中国民航报》《中国民用航空》杂志制作了新闻网站。 2002年9月，易航推出了校内第一个论坛——ehangbbs，受到了校领导的关怀与支持。
